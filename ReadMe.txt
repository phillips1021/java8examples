Java 8 Examples

Simple examples derived from various Java 8 tutorials.  Standard Maven
Java project.

Use mvn -e clean package to build the .jar file

cd to the target folder

To run one of the classes that have a main method (classes that have a
main method will end in App - for example FunctionApp)

java -cp java8examples.jar brucephillips.java8examples.FunctionApp
